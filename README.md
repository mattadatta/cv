# Curriculum Vitae

The CV of Matthew C. Brown.

Written in TypeScript with React and TailwindCSS.

## Running

Assuming one of `npm`/`pnpm`/`yarn`/etc. is already installed:

`pnpm install`

`pnpm run dev`

Navigate to the URL shown to you by Vite.
